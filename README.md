## 📌 توضیحات پروژه
یک اپلیکیشن ساخته شده با **React 19** که به شما اجازه می‌دهد شخصیت‌های سریال محبوب **Rick and Morty** را جستجو و بررسی کنید.  
این پروژه از **React Query** برای مدیریت درخواست‌ها، **TailwindCSS** برای استایل‌دهی و Context API برای مدیریت علاقه‌مندی‌ها استفاده می‌کند.

---

## ✨ ویژگی‌ها
- 🔍 جستجو بین شخصیت‌ها
- 📜 مشاهده جزئیات و وضعیت زنده/مرده بودن
- 🎬 نمایش لیست اپیزودهای هر شخصیت
- ❤️ اضافه و حذف از علاقه‌مندی‌ها (LocalStorage)
- ⚡ عملکرد سریع و کش هوشمند با React Query
- 📱 طراحی واکنش‌گرا

---
